# Claude Code 알고리즘 연습

Claude Code가 자주 출제되는 유형의 알고리즘 문제를 출제하고, 직접 풀어보는 공간입니다.

## 진행 방식

1. Claude Code에게 문제 출제를 요청
2. 유형별 폴더에 풀이 파일 생성
3. `node` 명령어로 바로 실행하여 정답 확인
4. 풀이 완료 후 Claude Code에게 코드 리뷰 요청

### 출제 규칙

- 테스트케이스는 최소 3개 제공 (기본 케이스, 간단한 케이스, 엣지 케이스 등)
- 문제 주석에 난이도 표시 (Lv.1 ~ Lv.5)

### 코드 리뷰 기준

풀이를 공유하면 아래 관점으로 리뷰합니다.

1. **정확성** - 정답 여부, 엣지 케이스 처리
2. **시간/공간 복잡도** - Big-O 분석, 더 효율적인 방법 제안
3. **코드 품질** - 변수명, 가독성, 불필요한 로직 여부
4. **개선 포인트** - 놓치기 쉬운 함정, 같은 유형에서 응용할 수 있는 팁

## 폴더 구조

```
Claude/
├── BFS/
├── DFS/
├── DP/
├── 그리디/
├── 이분탐색/
├── 투포인터/
├── 구현/
└── ...
```

## 실행 방법

```bash
node --watch algorithm/Claude/BFS/미로탈출_Lv2.js
```
